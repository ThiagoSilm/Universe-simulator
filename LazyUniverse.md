# Lazy Universe

## Overview
The Lazy Universe simulation provides an insightful understanding of cosmic structures and dynamics.

## Features
- **Realistic astrophysical models**
- **User-friendly interface**
- **Detailed visualizations**

## Getting Started
1. Ensure prerequisites are installed.
2. Clone the repository.
3. Run the simulation using the command `python lazy_universe.py`.
